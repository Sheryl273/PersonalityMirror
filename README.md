# AI Personality Mirror

🧠 Analyze your personality through AI-powered text analysis and visualize your traits with an interactive holographic interface.

## ✨ Features

- **Big Five Personality Analysis**: Comprehensive trait analysis (Openness, Conscientiousness, Extraversion, Agreeableness, Neuroticism)
- **Dynamic Avatar**: Personality-responsive avatar with expressions and animations
- **Interactive Visualizations**: Radar charts and timeline tracking
- **Real-time Analysis**: Instant personality feedback from text input
- **Premium UI**: Modern glass-morphism design with smooth animations

## 🛠️ Tech Stack

**Frontend**: React 18 + TypeScript, Vite, Tailwind CSS, Framer Motion, Chart.js

**Backend**: FastAPI, Hugging Face Transformers, Scikit-learn, MongoDB

**AI/ML**: Sentiment analysis, Big Five trait prediction, 7 emotional states
